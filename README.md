Student Record System

A beginner-friendly Python project that demonstrates how to store and manage student records using dictionaries and user input.

📌 Description

This program allows users to:

Add student records
Store student age
Store student city
Store student class
Search for students
View student details
Exit the program

Each student's information is stored inside a nested dictionary.

🧠 Concepts Used

Dictionaries
Nested Dictionaries
User Input
String Methods (strip(), lower())
Conditional Statements (if)
continue
while True loop
Search Operations
Data Validation

💻 Code Features

✅ Add Student Records

✅ Store Multiple Details Per Student

✅ Search Students by Name

✅ Count Total Students
